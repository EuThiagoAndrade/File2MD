# Política de Segurança

## Versões Suportadas

Atualmente, apenas a versão mais recente na branch `main` recebe atualizações de segurança.

| Versão | Suportada |
| :--- | :--- |
| `main` | ✅ Sim |
| Anteriores | ❌ Não |

## Como Reportar uma Vulnerabilidade

Por favor, **NÃO** abra uma Issue pública no GitHub para relatar problemas de segurança.

Se você descobrir uma falha de segurança neste projeto, por favor nos avise enviando um e-mail diretamente para:
**contato@euthiagoandrade.com.br**

Todas as vulnerabilidades de segurança serão prontamente avaliadas. Tentaremos responder ao seu e-mail em até 48 horas para alinhar os próximos passos e o cronograma de correção.
